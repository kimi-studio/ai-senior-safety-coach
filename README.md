# 안심하루 - AI 생활 코치 및 안전 알림 서비스

중장년층을 위한 AI 기반 생활 코치 및 안전 알림 앱 프로토타입

## 프로젝트 개요

**안심하루**는 중장년층(55~72세)이 매일 쉽게 사용할 수 있는 AI 생활 코치 앱입니다. 복약 관리, 일일 체크인, 안전 알림 기능과 보호자 안심 연동을 결합하여, 사용자는 건강한 일상을 유지하고 보호자는 안심할 수 있는 서비스를 제공합니다.

### 핵심 가치

- **실천 중심**: 건강 기록이 아닌 '오늘 바로 할 수 있는 행동' 제안
- **안심 경험**: 감시가 아닌 안심을 주는 보호자 연동
- **중장년 최적화**: 큰 글씨, 큰 버튼, 단순한 흐름의 UX

## 주요 기능

### 사용자 앱 (본인)
- **홈**: 오늘의 할 일과 AI 코칭 요약
- **AI 코칭**: 개인 맞춤 생활 루틴 제안
- **체크인**: 컨디션 기록 및 증상 관리
- **복약 관리**: 알림 및 복약 기록
- **도움 요청**: 위급 상황 대응 및 가족 연락
- **가족 연동**: 보호자와의 공유 설정
- **내 기록**: 주간 달성률 및 칭찬
- **설정**: 접근성 및 개인정보 관리

### 보호자 앱
- **오늘 상태 요약**: 체크인, 복약, 위험 이벤트 현황
- **활동 이력**: 최근 활동 타임라인
- **알림 설정**: 공유 범위 및 민감도 조절

### 발표용 페이지 (`/presentation`)
- 서비스 개요 및 문제 정의
- 타깃 사용자 페르소나
- 정보 구조 (IA) 및 핵심 플로우
- UX 설계 원칙
- 비즈니스 모델 및 확장 로드맵

## 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS 4
- **UI 컴포넌트**: Custom components (Button, Card, etc.)
- **아이콘**: Lucide React

## 시작하기

```bash
npm run dev
```

```bash
npm run build
```

```bash
npm start
```

## 주요 화면 경로

### 사용자 경험
- `/` - 웰컴 화면
- `/onboarding` - 온보딩 (기본 정보)
- `/onboarding/caregiver` - 온보딩 (보호자 설정)
- `/app` - 홈 (메인 화면)
- `/app/coaching` - AI 코칭 상세
- `/app/checkin` - 오늘의 체크인
- `/app/medication` - 복약 관리
- `/app/help` - 도움 요청 센터
- `/app/family` - 가족 연동 설정
- `/app/report` - 주간 리포트

### 보호자 경험
- `/caregiver` - 보호자 홈
- `/caregiver/history` - 활동 이력
- `/caregiver/settings` - 알림 설정

### 발표용
- `/presentation` - 한국어 서비스 기획 발표 페이지

## UX 설계 원칙

### 중장년 친화 UX
- 큰 텍스트 (18px ~ 32px)
- 큰 터치 영역 (최소 44px)
- 한 화면 한 가지 핵심 행동
- 생활 언어 사용 (전문 용어 배제)
- 아이콘 + 텍스트 중복 표현

### AI UX
- 선택형 코칭 우선 (긴 대화 지양)
- 추천 이유 짧게 설명
- 부담 없는 톤앤매너
- 실천 가능한 작은 행동 제안

### 안전 UX
- 공포가 아닌 안심 중심
- 단계적 알림 (즉시 → 재알림 → 보호자)
- 사용자 통제 가능한 공유 범위

```
│   │   ├── checkin/          # 체크인
│   │   ├── coaching/         # AI 코칭
│   │   ├── family/           # 가족 연동
│   │   ├── help/             # 도움 요청
│   │   ├── medication/       # 복약 관리
│   │   ├── notifications/    # 알림
│   │   ├── report/           # 리포트
│   │   └── settings/         # 설정
│   ├── caregiver/            # 보호자 앱 화면
│   │   ├── history/          # 활동 이력
│   │   └── settings/         # 알림 설정
│   ├── onboarding/           # 온보딩
│   ├── presentation/         # 발표용 페이지
│   └── page.tsx              # 웰컴 화면
├── components/               # 공용 컴포넌트
│   ├── ui/                   # UI 기본 컴포넌트
│   ├── app-header.tsx        # 앱 헤더
│   └── bottom-nav.tsx        # 하단 네비게이션
└── lib/                      # 유틸리티 및 데이터
    ├── mock-data.ts          # 목업 데이터
    └── utils.ts              # 헬퍼 함수
```

## 타깃 사용자

### Primary Persona - 본 사용자
- **연령**: 55~72세
- **특징**: 스마트폰 사용 가능하지만 복잡한 앱은 부담, 건강 관리 필요성 인지
- **니즈**: 쉬운 할 일 확인, 실천 가능한 조언, 위급 시 빠른 도움

### Secondary Persona - 보호자
- **연령**: 40~60대
- **특징**: 부모 건강 걱정, 상시 확인 불가, 과도한 알림 피하고 싶음
- **니즈**: 상태 요약, 중요 상황만 알림, 빠른 연락 수단
